# Solidarité Gabon — version multi-pages

Version organisée en vraies pages HTML.

## Pages
- `index.html` : Accueil
- `bureaux.html` : Nos bureaux
- `histoire.html` : Notre histoire
- `temoignages.html` : Témoignages
- `rejoindre.html` : Nous rejoindre
- `admin.html` : Espace admin

## Dossiers
- `css/styles.css` : styles personnalisés
- `js/app.js` : logique JavaScript commune
- `js/tailwind.config.js` : couleurs Tailwind

## Ouvrir avec Visual Studio Code
1. Dézippez le dossier.
2. Ouvrez le dossier dans VS Code.
3. Installez l’extension **Live Server**.
4. Clic droit sur `index.html` puis **Open with Live Server**.

Connexion admin démo : `admin` / `admin`.
